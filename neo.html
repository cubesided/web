cubeside//d Neo


cubesided Neo is a project that I am working on for my iSTEM Assessment Task 3. It is a helmet that is based on Halo 4 Master Chief design. It includes a HUD (Heads-Up Display) inside the helmet that acts as a vr headset where I can add the necessary UI ontop.

Coming Soon, around November 2025.



Info & Specs:

  • cubesided OSr1 Debian
  • Raspberry Pi 5
  • 10.1" Display - Waveshare Module
  • Custom 3D printed keyboard
